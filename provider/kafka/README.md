# Kafka Capability Provider
This capability provider is an implementation of the `wasmcloud:messaging` contract. It exposes publish, request, and subscribe functionality to actors.

## Installing Kafka
To install
```
brew install kafka
```

To start the Kafka service on a mac, use
```
brew services start zookeeper
brew services start kafka
```
To create a topic, use `make create-topic`, it's just for testing tho
